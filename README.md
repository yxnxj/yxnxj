:mega: Intro
------------------------
안녕하세요. **Spring 백엔드 개발자를 희망하는 조연준**입니다.

다른 영역의 사람들과 소통하는 것을 즐기며, 협업을 통해 보다 높은 퀄리티의 서비스를 제공하는 것에 초점을 맞춥니다.
 
많은 모습을 가진 사람들에게 제가 만든 서비스를 제공하는 것이 궁극적인 개발자로서의 목표이며, 
아직 세상에 존재하지 않는 서비스를 고민합니다.
